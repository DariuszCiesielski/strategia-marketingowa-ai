# Faza 3: Strategia

## Cel

Zdefiniowanie pozycjonowania marki, kluczowych przekazów i doboru kanałów marketingowych.

## Czas realizacji: 5-7 dni

## Krok 1: Pozycjonowanie

### Model pozycjonowania

Wypełnij poniższy framework:

**Dla** [segment klientów]
**którzy** [potrzeba/problem]
**nasza firma** [nazwa]
**jest** [kategoria]
**która** [kluczowa korzyść]
**w odróżnieniu od** [główna konkurencja]
**ponieważ** [dowód/powód do wiary]

### Przykład

**Dla** małych firm usługowych (5-20 osób)
**którzy** tracą czas na ręczne działania marketingowe
**nasza firma** MarketingBot
**jest** agencją AI marketingu
**która** automatyzuje 80% powtarzalnych zadań marketingowych
**w odróżnieniu od** tradycyjnych agencji pobierających stałe opłaty za ręczną pracę
**ponieważ** wykorzystujemy sprawdzone narzędzia AI, obniżając koszty o 60% przy zachowaniu jakości

## Krok 2: Persony klientów

Zdefiniuj 2-3 persony (nie więcej na start):

### Szablon persony

```
Imię: [fikcyjne, ale realistyczne]
Wiek: [przedział]
Stanowisko: [rola w firmie]
Firma: [typ, wielkość, branża]

Cele:
- [cel biznesowy 1]
- [cel biznesowy 2]

Frustracje:
- [problem 1]
- [problem 2]

Gdzie szuka informacji:
- [kanał 1 — np. LinkedIn, branżowe portale]
- [kanał 2 — np. YouTube, podcasty]

Proces decyzyjny:
- Kto podejmuje decyzję?
- Ile trwa cykl zakupowy?
- Jakie obiekcje ma przed zakupem?

Komunikacja, która działa:
- [typ treści, ton, format]
```

## Krok 4: Dobór kanałów

### Matryca kanałów

Oceń każdy kanał w skali 1-5:

| Kanał | Obecność klientów | Koszt | Czas do efektu | Skalowalność | **Suma** |
|-------|------------------|-------|----------------|-------------|---------|
| SEO + Blog | | | | | |
| Google Ads | | | | | |
| Facebook/Instagram | | | | | |
| LinkedIn | | | | | |
| YouTube | | | | | |
| E-mail marketing | | | | | |
| TikTok | | | | | |
| Partnerstwa | | | | | |

### Rekomendacje per typ firmy

| Typ firmy | Kanały priorytetowe | Kanały wspierające |
|-----------|-------------------|-------------------|
| B2B usługi | LinkedIn, SEO, e-mail | Google Ads, YouTube |
| E-commerce | Google Ads, Instagram, SEO | E-mail, Facebook, TikTok |
| Lokalna firma | Google Maps, Facebook, SEO lokalne | Instagram, e-mail |
| SaaS | SEO, content marketing, LinkedIn | Google Ads, e-mail, partnerstwa |

## Krok 5: Budżet strategiczny

### Podział budżetu (reguła 70-20-10)

- **70%** — sprawdzone kanały (to co już działa)
- **20%** — nowe kanały z potencjałem (eksperyment)
- **10%** — innowacje (AI narzędzia, nowe formaty)

### Przykładowy budżet 3000 zł/mies.

| Pozycja | Kwota | Cel |
|---------|-------|-----|
| Google Ads | 1200 zł | Leady z wyszukiwania |
| Meta Ads (FB/IG) | 600 zł | Remarketing + brand |
| Narzędzia AI | 300 zł | ChatGPT Pro + Canva Pro |
| Mailerlite | 100 zł | Nurturing leadów |
| Stock zdjęcia/wideo | 200 zł | Materiały do treści |
| Eksperyment (TikTok Ads) | 400 zł | Test nowego kanału |
| Rezerwa | 200 zł | Nagłe potrzeby |
