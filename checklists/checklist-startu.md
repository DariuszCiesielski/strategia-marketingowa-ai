# Checklist startu kampanii marketingowej

Przed uruchomieniem jakichkolwiek działań marketingowych, przejdź przez tę listę.

## Fundamenty

- [ ] Audyt marketingowy przeprowadzony ([szablon](../templates/audyt-template.md))
- [ ] Cele SMART zdefiniowane (min. 3)
- [ ] Persony klientów opisane (min. 2)
- [ ] Pozycjonowanie marki sformułowane
- [ ] Budżet zaplanowany i zatwierdzony
- [ ] Osoba odpowiedzialna za marketing wyznaczona

## Strona internetowa

- [ ] Strona responsywna (przetestuj na telefonie)
- [ ] SSL aktywny (https://)
- [ ] Czas ładowania <3 sekundy (sprawdź: PageSpeed Insights)
- [ ] Meta tagi (title + description) na kluczowych stronach
- [ ] Google Analytics zainstalowany i działający
- [ ] Google Search Console skonfigurowane
- [ ] Formularz kontaktowy działa (wyślij test)
- [ ] Polityka prywatności i RODO
- [ ] Cookie banner skonfigurowany
- [ ] Strona "O nas" aktualna
- [ ] Dane kontaktowe widoczne (telefon, e-mail, adres)
- [ ] CTA (call-to-action) na każdej ważnej stronie
- [ ] UTM-y przygotowane dla kampanii

## Social media

- [ ] Profile kompletne (zdjęcie, bio, link do strony)
- [ ] Szablony graficzne przygotowane (Canva brand kit)
- [ ] Minimum 5 postów zaplanowanych na przód
- [ ] Narzędzie do schedulowania skonfigurowane
- [ ] Ton komunikacji ustalony (formalny/nieformalny/ekspercki)
- [ ] Hashtagi zbadane i zapisane (per kanał)
- [ ] Plan odpowiedzi na komentarze (kto, kiedy, szablon)

## E-mail marketing

- [ ] Narzędzie skonfigurowane (Mailerlite/ConvertKit/inne)
- [ ] Formularz zapisu na stronie (z lead magnetem)
- [ ] E-mail powitalny (welcome) gotowy
- [ ] Szablon newslettera przygotowany
- [ ] Segmenty zdefiniowane (min.: nowi, kupujący, VIP)
- [ ] Adres nadawcy zweryfikowany
- [ ] SPF/DKIM skonfigurowane (dostarczalność)
- [ ] Test wysyłki (sprawdź spam folder)
- [ ] Link do wypisania się działa

## Reklamy płatne (jeśli planowane)

- [ ] Konto reklamowe skonfigurowane
- [ ] Pixel/tag zainstalowany na stronie
- [ ] Konwersje zdefiniowane (formularz, zakup, telefon)
- [ ] Grupy docelowe stworzone
- [ ] Kreacje przygotowane (min. 3 warianty)
- [ ] Landing page gotowy i przetestowany
- [ ] Budżet dzienny ustawiony (z limitem)
- [ ] Plan testów A/B (co testujemy w pierwszym tygodniu)

## Automatyzacje

- [ ] Workflow zaprojektowane i opisane
- [ ] Integracje podłączone (API, webhooks)
- [ ] Testy z danymi testowymi przeprowadzone
- [ ] Alerty na awarie ustawione
- [ ] Fallback plan dla każdego workflow

## Treści

- [ ] Filary treści zdefiniowane (3-5)
- [ ] Bank tematów na 30 dni (min. 10 tematów)
- [ ] Kalendarz redakcyjny wypełniony (tydzień 1-2)
- [ ] Szablony promptów AI przygotowane
- [ ] Proces review treści ustalony (kto sprawdza przed publikacją)
- [ ] Bank zdjęć/grafik (min. 20 materiałów)

## Monitoring i raportowanie

- [ ] Dashboard KPI skonfigurowany
- [ ] Częstotliwość raportowania ustalona
- [ ] Szablon raportu gotowy ([szablon](../templates/raport-miesięczny.md))
- [ ] Google Alerts na nazwę firmy
- [ ] Monitoring konkurencji ustawiony

## Prawo i compliance

- [ ] Regulamin strony/sklepu aktualny
- [ ] Polityka prywatności zgodna z RODO
- [ ] Zgody marketingowe zbierane poprawnie (double opt-in)
- [ ] Klauzula informacyjna w formularzach
- [ ] Oznaczanie treści sponsorowanych/reklamowych

---

**Gotowość do startu**: Wszystkie checkboxy w sekcjach "Fundamenty", "Strona internetowa" i minimum jednego kanału muszą być zaznaczone.

Opcjonalne sekcje (reklamy, automatyzacje) — uzupełnij przed uruchomieniem danego elementu.
